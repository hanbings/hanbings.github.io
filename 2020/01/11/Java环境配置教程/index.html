Java环境配置教程

Java环境配置教程

这个教程针对使用Java开发的环境部署（JDK 1.8）

1.下载Java的安装包

下载 JDK（ Java Development ToolKit，Java语言开发工具包 ）

下载地址：

​ Java官网下载

​ JDK 32位 腾讯软件中心

​ JDK 64位 腾讯软件中心

2.安装JDK

下载JDK的安装包后双击运行

可以更改安装路径，但请确保知道安装路径到底在哪里

如果真的是萌新请一路Next下一步就行 =.=

3.为Java添加环境变量

使用Windows的搜索功能找到编辑环境变量

Windows7版本的也可以右键单击计算机图标打开系统属性找到高级配置

单击”新建“

然后添加

变量名：JAVA_HOME

变量值：C:\Program Files\Java\jdk1.8.0_191

注意：变量值是安装Java的路径，如果前面改了安装路径请填写更改后的安装路径，不同版本的Java路径不同，请不要直接复制

点击确定，再次“新建”

变量名：CLASSPATH

变量值：.;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar;

注意：请完整复制，包括.和；

点击确定，在列表中找到Path，选中后点击编辑

点击新建：%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin;

一路确定保存

4.验证环境已经部署好

同时按下Win和R键

在“运行”窗口内输入cmd

在“CMD”窗口内输入java -version

注意：java -version java和-version之间有空格

如果返回是

Java Version: JDK-xxx.xxx.xxx

或类似的字符串那么恭喜你，Java环境已经部署完成

如果返回：“Java不是内部程序”这样的字样，请再仔细检查每一步

END.
