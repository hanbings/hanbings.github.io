水一篇安装LNMP环境

打开LNMP网站然后把指令贴到ssh里面

https://lnmp.org/install.html 找到

wget http://soft.vpser.net/lnmp/lnmp1.7.tar.gz -cO lnmp1.7.tar.gz && tar zxf lnmp1.7.tar.gz && cd lnmp1.7 && ./install.sh lnmp

有一些安装前的设置

  1. 选择数据库 就Mysql和MariaDB两种 因为我要安装Jfrog Artifactory所以选择了Mysql 5.7 选择数据库后会设置一个数据库密码 如果不输入将随机一个 记得注意看看
  2. 如果选的是Mysql会询问是否需要启用MySQL InnoDB，MySQL 5.7+版本无法关闭InnoDB，这个看自己
  3. 选择PHP版本 如果是从Lsky-Pro那边来的选7.4就行啦~ 如果有新版本也可以试试新版本
  4. 然后会有一个内存调度器 个人使用应该没有很大区别 我选了TCMalloc 不选择默认的是不安装
  5. 然后按下随意一个键就可以安装了 这个时候返回可以选择Ctrl+c

挺水的是吧 ：D
